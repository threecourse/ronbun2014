Pythonによる選択効果の分析

Pythonによって、選択効果の分析を仮想データで行ってみました。

行ったこと
０．仮想の保有データ・支払データを生成
  契約データを人口の1/1000、粗死亡率を標準生命表、選択効果を経過別に40-90%として、乱数によりデータを生成しています。
１．年齢群団・性別ごと、経過・性別ごとに集計して選択効果を計算
２．以下のグラフを作成
  a. 選択効果
  b. 選択効果、95%信頼区間を表示
  c. 粗発生率を基準とした死差益、95%信頼区間を表示
  それぞれ、年齢群団別・性別、経過別・性別で作成。
  信頼区間は保有Nと実績支払率Sから求めました。（あまり厳密ではないと思います）
